MAQUETTE

The object held in hand, scale appropriate for ritual use and icon worship.

River Idol maquette on site in the green space of Jones Confluence State Park.

The confluence viewed from above, with the Missouri riveron the left and the Mississippi on the right.

Unlike a "typical" bronze monument, which is often large and permanently installed in public view, River Idol fits in the hand and is intended to be used as a token of worship, buried on site or used in ritual. This makes installation maquettes challenging to render, as the installation itself is much more personal than grandiose, based on independent practices of folk religiosity.

River Idol can function at a larger scale, as shown in the site-specific maquette. A larger, more traditional installation like this would be more appropriate for a formal proposal to local government than the smaller icon size. The scale, of course, changes the meaning and function of the work. The more intimate, smaller piece functions best when activated by community particpation. The larger work is more static, activated by the environment and available for engagement from random passerby. Knowledge of the context is not needed to engage with the larger work.

As a single object, its installation is determined by the user and highly variable; as an object that can be 3D-printed en masse, the installation capabilites can be even more varied.
